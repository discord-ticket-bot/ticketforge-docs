    Welcome to TicketForge

    TicketForge is the ultimate Discord ticketing bot designed to streamline support requests, automate workflows, and organize your community interactions.

    Whether you are running a small gaming community or a large enterprise server, TicketForge allows you to manage support requests through an intuitive web dashboard.

    Key Features

    Feature Description
    Intuitive Creation Allow users to open tickets via customizable Buttons or Select Menus.
    Custom Forms Collect essential data before a ticket opens using interactive Modal Forms.
    Smart Automation Automate Auto-Closing, User Leave Handling, and custom Auto-DMs.
    Claiming System Assign specific staff to tickets to prevent collisions and track performance.
    Rating System (CSAT) Gather user feedback with 1-5 star ratings and a public Review Feed.
    Archiving & Logging Generate permanent HTML Transcripts and real-time Audit Logs of all events.
    Schedules Set Operational Hours and timezones to control when your team is available.
    Complete Customization Full control over Embed Colors, Author Icons, and Message Variables.
